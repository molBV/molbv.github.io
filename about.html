About Aviar

Aviar is a lightweight, fast, browser-native arcade game built for quick sessions and surprising depth (boss phases, upgrades, secrets).

Built with help from ChatGPT — From early prototypes to content pages and polish passes, we used ChatGPT to accelerate ideas, drafts, and code sketches. The game code is hand-tuned; the process was AI-assisted.

Tech & Design

Aviar uses a lean HTML5 canvas loop, minimal allocations, and hand-tuned collision to keep input latency low. Assets are optimized for instant play.

Why no ads on the gameplay page?

We keep ads off the canvas page to prevent accidental clicks and preserve flow. Ads will appear only on help, blog, and info pages after approval.
